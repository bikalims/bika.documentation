### bika.documentation

bika.documentation Is a repository where some of the documentation for bika is stored and maintained

Currently only the full [server installation manual](https://github.com/bikalims/bika.documentation/blob/main/docs/BikaSenaiteServerIntroduction.md). For the Docker installation see [Ingwe Bika LIMS 4 - Docker Suite](https://github.com/bikalims/bika.lims/blob/main/README.md)

**Part of the Bika LIMS Open Source Project**  
[www.bikalims.org](https://www.bikalims.org)

Bika is an Open Soure Laboratory Management System (LIMS). It supports ISO 17025 and is used by labs around the world for sample tracking, analytical workflows, instrument integration, QC, results reporting, invoicing and ISO compliance.
